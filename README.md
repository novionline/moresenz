# MoreSenz website

NectarBlocks driven WordPress website 🚀 

The application runs on the following software:

> Please use these versions to prevent incompatibility issues.

- PHP 8.4
- MySQL 8.0
- Node 22

## Set up Valet

- ```$ cd public```
- ```$ valet link moresenz```
- ```$ valet secure```
- ```$ valet use php@8.4```

Create your local database called: ```moresenz_loc```

Your development environment should now be available at ```https://moresenz.test```
